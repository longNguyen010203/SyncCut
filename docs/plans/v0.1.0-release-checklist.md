## Release Notes

SyncCut v0.1.0 is the first MVP release candidate for the prepared-input-to-Remotion-render pipeline. It turns prepared scene, narration audio, and alignment inputs into a structured timeline that can be inspected, validated, exported to Remotion props, prepared with public assets, checked with preflight, and rendered locally through the Remotion project.

The MVP pipeline includes timeline building from `scenes.json`, section audio files, and alignment JSON files; timeline validation and inspection; Remotion props export; public audio asset preparation; optional local AI_VIDEO/B_ROLL visual asset preparation; visual readiness inspection; props-only and public-file preflight checks; and local Remotion still, smoke, segment, and final render workflows. The Remotion workflows are driven by npm scripts under `remotion/`, not by Python commands invoking Remotion.

The TSMC sample was validated after the timing and visual-duration polish.
